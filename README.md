Bash, Pipes, & Socket SDK
=========================

> This repository is a companion to the
> [Bash, Pipes, & Socket SDK][blog] blog post.

## Running the Application

```bash
ssc compile -r -o .
```

Or with [bpkg](https://github.com/bpkg/bpkg):

```bash
bpkg run start
```
